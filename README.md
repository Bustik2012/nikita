<h2>Maze generator</h2>
This is a very simple maze generator for minecraft java 1.16.5 . It consists of a "natura" generated maze (which means that it isnt a straight hallway that goes through 500 blocks, but instead a randomly chosen width is used) and multiple floors after meeting a certain condition later on
